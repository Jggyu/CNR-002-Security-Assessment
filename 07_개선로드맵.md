# CNR-002 OT/드론/SCM 망 보안 개선 로드맵

## 📊 현황 요약
- **전체 준수율**: 69% (102/148 항목 준수)
- **핵심 취약점**: 망분리체계 고도화, AI보안 강화, 드론통신 보안
- **총 투자 규모**: 약 36.5억원 (3년간)
- **개선 목표**: 95% 준수율 달성

## 🚀 3단계 개선 계획

### Phase 1: 즉시 개선 (2025년 3분기, 3개월)
> **목표**: 상위험 취약점 해결 및 기본 보안체계 강화

#### 🎯 주요 과제
1. **드론-GCS 상호인증 시스템 구축**
   - TLS 1.3 기반 상호인증 적용
   - 예상비용: 3억원 | 담당: IT/보안팀

2. **LLM 보안모듈 개발**
   - 프롬프트 인젝션 방어
   - AI API 보안 강화
   - 예상비용: 5억원 | 담당: 연구개발/IT팀

3. **망간연동 보안 강화**
   - 드론-OT망 물리적 분리
   - 전용 보안장비 도입
   - 예상비용: 8억원 | 담당: IT/보안팀

4. **공급망 보안 강화**
   - OT장비 공급업체 보안인증 의무화
   - 예상비용: 1억원 | 담당: 구매/보안팀

#### 📈 기대효과
- 준수율: 69% → 82%
- 상위험 취약점: 6개 → 2개
- 투자비용: 17억원

---

### Phase 2: 단기 개선 (2025년 4분기, 3개월)
> **목표**: 중위험 해결 및 통합 관제 체계 구축

#### 🎯 주요 과제
1. **FHSS 고도화 및 안티재밍 강화**
   - 주파수 도약 기술 완전 적용
   - 전자전 대응 능력 향상
   - 예상비용: 4억원 | 담당: 연구개발팀

2. **MILS 이중화 구성**
   - 망연계장비 이중화
   - 고가용성 확보
   - 예상비용: 3억원 | 담당: IT팀

3. **OT 패치관리 자동화**
   - OT 전용 패치관리시스템
   - 생산 중단 최소화
   - 예상비용: 1.5억원 | 담당: IT팀

4. **AI 데이터 보안 강화**
   - 동형암호화 기술 도입
   - 학습데이터 보안 향상
   - 예상비용: 3억원 | 담당: 연구개발팀

#### 📈 기대효과
- 준수율: 82% → 90%
- 중위험 취약점: 5개 → 2개
- 투자비용: 11.5억원

### Phase 3: 중기 개선 (2026년 상반기, 6개월)
> **목표**: 선진 보안체계 완성 및 미래 대비

#### 🎯 주요 과제
1. **통합 보안관제센터 구축**

   - 예상비용: 6억원 | 담당: IT/보안팀

   - 네트워크 신뢰 모델 전환
   - 마이크로 세그멘테이션
   - 예상비용: 8억원 | 담당: IT/보안팀
